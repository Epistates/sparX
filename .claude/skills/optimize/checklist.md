# Post Optimization Checklist

Run every draft through this checklist before publishing.

## Reach Killers (any = immediate fix)
- [ ] No external links in post body
- [ ] No engagement bait phrases ("follow for more", "like if you agree", "RT this")
- [ ] No more than 2 hashtags
- [ ] No self-reply link chains
- [ ] Not the 4th+ post today without proportional engagement

## Hook Quality
- [ ] First 8-12 words create curiosity, tension, or promise value
- [ ] Hook is accessible (not pure jargon — even experts appreciate clarity)
- [ ] Hook matches the actual content (no bait-and-switch)
- [ ] Would YOU stop scrolling for this?

## Dwell Time Optimization
- [ ] Line breaks create visual breathing room
- [ ] Post contains at least one specific number or proof point
- [ ] Content is insight-dense (every sentence adds value)
- [ ] Nothing can be cut without losing meaning
- [ ] Formatting uses short sentences / fragments for rhythm

## Reply Generation
- [ ] Ends with a genuine question or debatable take
- [ ] Question is specific enough to answer (not "thoughts?")
- [ ] Content has a natural "yes, but..." or "for me it's..." response angle
- [ ] Doesn't feel like it's fishing for engagement

## Bookmark Appeal
- [ ] Contains actionable information someone would reference later
- [ ] Specific enough to be useful (not just motivation/vibes)
- [ ] Would you save this if someone else posted it?

## Negative Signal Prevention
- [ ] Content is on-topic for the author's niche
- [ ] No needlessly antagonistic framing
- [ ] Hook promise matches delivery (no disappointment trigger)
- [ ] Won't trigger "not interested" from adjacent audiences

## Voice & Authenticity
- [ ] Reads like a human wrote it (not corporate/ChatGPT-generic)
- [ ] Author's personality comes through
- [ ] Claims are backed by evidence or experience
- [ ] Not trying too hard to be clever or provocative

## Media (if applicable)
- [ ] Image is high-quality and readable on mobile
- [ ] Video is under 45 seconds (ideally 15-30s)
- [ ] Video hooks in first 3 seconds (no intros)
- [ ] Alt text included for images
- [ ] Media has supporting text (not media-only post)

## Thread-Specific (if thread)
- [ ] Each tweet stands alone with value
- [ ] No numbering (1/, 2/, etc.)
- [ ] 5-7 tweets (sweet spot)
- [ ] At least one visual in middle tweets
- [ ] Links only in a reply after the thread
- [ ] Each tweet under 280 characters
