# Thread Structures — Proven Patterns

## The Tutorial Thread (5-7 tweets)

Best for: How-tos, step-by-step guides, setup instructions

```
Tweet 1: [HOOK] How to [achieve result] in [timeframe/steps] ↓
Tweet 2: [CONTEXT] Why this matters / the problem it solves
Tweet 3: [STEP 1] First action — specific and actionable
Tweet 4: [STEP 2-3] Core steps — include code/screenshot
Tweet 5: [STEP 4 + GOTCHA] Final step + common mistake to avoid
Tweet 6: [RESULT] What you get — screenshot/proof
Tweet 7: [CTA] Question + where to learn more
```

## The Deep Dive Thread (6-8 tweets)

Best for: Technical explorations, architecture decisions, analysis

```
Tweet 1: [HOOK] I spent [time] investigating [topic]. Here's what I found ↓
Tweet 2: [CONTEXT] Background — why this matters now
Tweet 3: [FINDING 1] Most surprising discovery
Tweet 4: [FINDING 2] Second key insight — with data/proof
Tweet 5: [FINDING 3] Third insight or counter-intuitive result
Tweet 6: [IMPLICATIONS] What this means for [audience]
Tweet 7: [TAKEAWAY] One-sentence summary + question
```

## The Story Thread (5-7 tweets)

Best for: Build-in-public, postmortems, journey narratives

```
Tweet 1: [HOOK] [Dramatic result or turning point] — here's how it happened ↓
Tweet 2: [SETUP] Where it started — the problem/situation
Tweet 3: [STRUGGLE] What went wrong / the hard part
Tweet 4: [TURNING POINT] The insight or breakthrough moment
Tweet 5: [RESOLUTION] What changed — specific results
Tweet 6: [LESSON] The takeaway that applies to the reader
Tweet 7: [CTA] What's your [similar experience]?
```

## The Listicle Thread (5-7 tweets)

Best for: Curated tips, resources, tools, observations

```
Tweet 1: [HOOK] [N] [things] that [specific benefit]. Most [audience] miss #3 ↓
Tweet 2: [ITEM 1] Most impactful item first (front-load value)
Tweet 3: [ITEM 2-3] Two more items (can combine shorter ones)
Tweet 4: [ITEM 4-5] Include the "most people miss this" item here
Tweet 5: [ITEM 6-7] Final items
Tweet 6: [BONUS] The "one more thing" — most unexpected/valuable
Tweet 7: [CTA] Which resonated most? What would you add?
```

## The Comparison Thread (5-7 tweets)

Best for: Tool comparisons, benchmark results, A vs B analysis

```
Tweet 1: [HOOK] [Tool A] vs [Tool B] — I tested both on [real workload]. Clear winner ↓
Tweet 2: [SETUP] What I tested, methodology, hardware
Tweet 3: [RESULTS] Side-by-side numbers — table or screenshot
Tweet 4: [ANALYSIS] Why the winner wins — technical explanation
Tweet 5: [NUANCE] Where the loser actually wins (fairness builds trust)
Tweet 6: [VERDICT] Clear recommendation with specific use cases
Tweet 7: [CTA] What's your experience? Running different benchmarks?
```

## The Contrarian Thread (5-7 tweets)

Best for: Challenging conventional wisdom, strong opinions, thought leadership

```
Tweet 1: [HOOK] Everyone says [common belief]. I think they're wrong. Here's why ↓
Tweet 2: [COMMON VIEW] Steel-man the opposing position (shows fairness)
Tweet 3: [COUNTER 1] Your strongest counterargument
Tweet 4: [COUNTER 2] Supporting evidence — data, experience, examples
Tweet 5: [COUNTER 3] The "but what about..." preemptive response
Tweet 6: [SYNTHESIS] The nuanced truth between the extremes
Tweet 7: [CTA] Provocative question that invites debate
```

## Thread Flow Techniques

### Micro-Hooks Between Tweets
End tweets with tension that pulls to the next:
- "But that's not the interesting part."
- "Then I discovered something unexpected."
- "Here's where it gets weird:"
- "The real lesson was the opposite of what I expected."

### Standalone Test
Every tweet must pass the "saw this in isolation" test:
- Would it make sense without the thread context?
- Would it deliver some value on its own?
- Would it make someone want to read the rest?

### Visual Placement
Place images/screenshots in tweets 3-5 (middle of thread):
- Early images compete with the hook
- Middle images re-engage readers who might drop off
- Screenshots of data, code, or results have highest dwell time

### Thread Length Decision Matrix

| Content Depth | Audience Knowledge | Recommended Length |
|--------------|-------------------|-------------------|
| Surface-level | Any | Don't thread — use a single post |
| Moderate | Beginner | 5-6 tweets with more context |
| Moderate | Expert | 4-5 tweets, dense |
| Deep | Beginner | 7-8 tweets, build up gradually |
| Deep | Expert | 6-7 tweets, assume knowledge |
| Exhaustive | Any | 8-10 max, or split into 2 threads |
